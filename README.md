# ShopForge 🛒

A modern, dark-themed e-commerce web app built with vanilla HTML, CSS, and JavaScript.

## Features
-  Product catalog with category filters
-  Product detail page with size/option selector
-  Shopping cart with quantity controls
-  Checkout with multiple payment methods (Card, UPI, COD, Net Banking)
-  User registration & login (localStorage-based)
-  Wishlist
-  Order history & tracking timeline
-  Promo code support (SAVE10)
-  Toast notifications
-  Responsive design

## Tech Stack
- Frontend: HTML5, CSS3, Vanilla JavaScript
- Storage: localStorage (no backend required)
- Fonts: Syne + DM Sans (Google Fonts)

## Getting Started
Just open `shopforge.html` in any browser — no build step needed.

## Future Scope
